A record of my journey learning fullstack web development — projects, notes, experiments, and progress along the way.

📖 About

This repository documents my path to becoming a fullstack developer. It contains the projects I build, concepts I practice, and milestones I hit as I learn frontend, backend, databases, and everything in between.

🎯 Goals


 Get comfortable with HTML, CSS, and JavaScript fundamentals
 Learn a frontend framework (React/Vue/Angular)
 Build and consume REST APIs
 Work with a backend framework (Node.js/Express, Django, etc.)
 Learn database design (SQL and/or NoSQL)
 Deploy a full application end-to-end
 Build a portfolio-ready capstone project
