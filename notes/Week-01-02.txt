Role of Semicolons in JavaScript

Semicolons are used to mark the end of statements in javascript.
let variableOne = 35; // Declare a variable with value of 35 and the semicolon marks the end of this statement.

If semicolons are omitted from JavaScript code they are added automatically by ASI(Automatic Semicolon Insertion).
But ASI can not be trusted because it might cause unexpected behaviour or errors in the code or program.
Therefore, it is always adviseable to add semicolons explicitly in your code after every statement to avoid such errors.
